# Hail Mary XR

A prototype for a WebXR using Unity.

## How to develop

```sh
# Run the local server which supports Brotli files for Unity WebGL
npm run serve

# Run ngrok if you want to test on a device, such as Meta Quest.
ngrok http 8080
```
